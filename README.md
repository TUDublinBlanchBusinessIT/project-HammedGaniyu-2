# Sneaker Collection Project

This is a simple PHP and MySQL project that lets me add sneakers to a database and view them on a display page.

## Features
- Add sneaker (Brand, Model, Colorway, Size)
- Dropdown for brand
- View all sneakers in the database

## Files
- index.php – add sneaker
- insert.php – save sneaker
- display.php – show sneakers
- db.php – database connection
- sql folder – database versions ie DBV1 DBV2 DBV3
